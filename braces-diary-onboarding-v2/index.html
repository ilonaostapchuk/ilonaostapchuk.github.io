Onboarding for the Braces Diary mobile app (Version #2)

1
13

Project

Designed a new multi-step onboarding flow that gradually introduces the app's benefits, collects relevant user information, encourages notification opt-in, and presents the premium subscription at the most relevant moment. The experience was optimized to be informative while maintaining a smooth and engaging user journey.

Role

Led the end-to-end redesign of the onboarding experience for the Braces Diary mobile app, including UX strategy, user flows, wireframes, UI design, and onboarding content structure.

Challenge

The existing onboarding was outdated and no longer provided enough value to new users. It lacked guidance, failed to collect essential user information, and did not effectively communicate the app's key features or premium offering.

Result

Created a more structured and user-focused onboarding experience that better educates users about the product, improves personalization through profile setup, increases the visibility of key features, and provides a stronger foundation for premium conversion and long-term engagement.
